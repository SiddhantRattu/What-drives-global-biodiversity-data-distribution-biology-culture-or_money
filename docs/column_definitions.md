# Column Definitions

## Colonizers
- `col.belgium`: Indicator for colonization by Belgium (1 = Yes, 0 = No)
- `col.britain`: Indicator for colonization by Britain
- `col.france`: Indicator for colonization by France
- `col.germany`: Indicator for colonization by Germany
- `col.italy`: Indicator for colonization by Italy
- `col.netherlands`: Indicator for colonization by Netherlands
- `col.portugal`: Indicator for colonization by Portugal
- `col.spain`: Indicator for colonization by Spain

## Language
- `main_language_code`: ISO code of the main language
- `is_english_main`: Boolean indicating if English is the main language
